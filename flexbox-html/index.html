Aplikace Flexbox a flex-direction

Pokud chceme použít na element flastnotsti Flexbox, pak je třeba do jeho CSS kódu napsat display: flex.

Toto nám umožňuje srovnat elementy uvnitř tohoto elementu do řad a sloupců pomocí flex-direction. Pokud napíšeme flex-direction: row pak zarovnáme elementy horizontálně. Pokud napíšeme flex-direction: column zarovnáme elementy vertikálně.

Lze také použít možnosti obrácené řady row-reverse nebo obráceného sloupce column-reverse.

Příklad:

.priklad {

display: flex;

height: 500px;

width: 500px;

background-color: grey;

}

 

.priklad-BOX1 {

background-color: blue;

width: 40%;

height: 40%;

}

 

.priklad-BOX2 {

background-color: red;

width: 40%;

height: 40%;

}

flex-direction: row;

Tak se nic nezmění. Toto je výchozí nastavení.

Justify-content

Pokud nejsou elementy roztaženy přes celý kontejner můžeme použít justify-content abychom je zarovnali a uzpůsobili místo mezi nimi.

Hlavní osa se řídí podle směru, dle kterého jsou elementy poskládány. Tedy pro řadu (row) je to vodorovná osa a pro sloupec (column) vertikální.

Obrázek pro vysvětlení:https://www.w3.org/TR/css-flexbox-1/images/flex-direction-terms.svg

  • flex-start: zarovná elementy na začátek flex kontejneru. (Pro row doleva, pro column nahoru.) Toto je výchozí hodnota.
  • flex-end: zarovná elementy na konec flex kontejneru. (Pro row doprava, pro column dolů.)
  • space-between: zarovná elementy na střed hlavní osy tak, že je mezi nimi stejná mezera. První a poslední element jsou na krajích kontejneru.
  • space-around: stejné jako space-between, ale první a poslední element nejsou na kraji, ale je na konci poloviční mezera, než mezi jednotlivými elementy.
  • space-evenly: mezery mezi elementy a na okrajích jsou stejné.

Align-items

align-items je podobné jako justify-content. Rozdíl je v tom, že se neřídí dle hlavní osy, ale dle příčné.

Příčná osa je opakem hlavní osy. Pro řady je vertikální a pro sloupce horizontální.

  • flex-start: zarovná elementy na začátek flex kontejneru. (Pro row nahoru, pro column doleva.)
  • flex-end: zarovná elementy na konec flex kontejneru. (Pro row dolů, pro column doprava.)
  • center: zarovná elementy na střed.
  • stretch: roztáhne elementy tak, aby zaplnili celý kontejner. Např. řady se roztáhnou tak, aby zaplnily kontejner odshora dolů. Toto je výchozí hodnota.
  • baseline: zarovná elementy dle účaří.

Flex-wrap

Pokud se nám všechny elementy nevlezou vedle sebe (nebo pod sebe v případě sloupce), pak můžeme nechat vytvořit další řádek (nebo sloupec) pomocí flex-wrap.

  • nowrap: nevytváří další řádek nebo sloupec. Toto je výchozí hodnota.
  • wrap: zalamuje zprava doleva nebo sloupce shora dolů.
  • wrap-reverse: zalamuje zleva doprava nebo sloupce zdola nahoru.

Flex-shrink

Pokud je kontejner příliš malý, aby se do něj elementy vlezly, můžeme použít flex-shrink. Tento nám umožní zmenšit elementy. Ke zmenšení dojde pokud jsou kombinované šířky elementů v kontejneru větší, než šířka kontejneru samotného.

flex-shrink používá čísla jako hodnoty. Čím větší je číslo, tím více se zmenší v porovnání s ostatními elementy v kontejneru. Tedy například pokud má jeden element hodnotu 1 a druhý hodnotu 3, tak se ten druhý zmenší třikrát více než ten první.

Příklad:

.priklad {

display: flex;

height: 500px;

width: 500px;

background-color: grey;

}

 

.priklad-BOX1 {

background-color: blue;

width: 60%;

flex-shrink: 1;

}

 

.priklad-BOX2 {

background-color: red;

width: 60%;

flex-shrink: 3;

}

Flex-grow

Pokud je kontejner příliš velký, můžeme použít flex-grow. Tento nám umožní zvětšit elementy. Ke zvětšení dojde pokud jsou kombinované šířky elementů v kontejneru menší, než šířka kontejneru samotného. flex-grow je opakem flex-shrink.

flex-grow (stejně jako flex-shrink) používá čísla jako hodnoty. Čím větší je číslo, tím více se zvětší v porovnání s ostatními elementy v kontejneru. Tedy například pokud má jeden element hodnotu 1 a druhý hodnotu 3, tak se ten druhý zětší třikrát více než ten první.

Příklad:

.priklad {

display: flex;

height: 500px;

width: 500px;

background-color: grey;

}

 

.priklad-BOX1 {

background-color: blue;

width: 30%;

flex-grow: 1;

}

 

.priklad-BOX2 {

background-color: red;

width: 30%;

flex-grow: 3;

}

Flex-basis

Pokud chceme určit velikost elementů před tím, než se použije zmenšení nebo zvětšení, pak můžeme použít flex-basis. Velikost určí tu, kde je hlavní osa, tedy pro řady určuje šířku a pro sloupce výšku.

Před height nebo width má přednost. Tedy určení šířky je zbytečné, pokud jsme určili flex-basis pro řadu a určení výšky je zbytečné, pokud jsme určily flex-basis pro sloupec.

flex-basis aplikujeme na elementy uvnitř kontejneru.

Zkrácený zápis

Vlastnosti flex-grow, flex-shrink a flex-basis v tomto pořadí lze zapsat i ve zkrácené formě pomocí vlastnosti flex tak, že jejich hodnoty napíšeme za sebe s mezerami.

Tedy tento kód:

flex-grow: 1;

flex-shrink: 0;

flex-basis: 10px;

lze zapsat takto:

flex: 1 0 10px;

Pořadí

Normálně se elementy ve flex kontejneru seřadí dle toho, jak jdou po sobě ve zdrojovém HTML kódu. Pokud jejich pořadí chceme změnit bez přeskládání v HMTL, pak použijeme vlastnost order.

Pořadí se určuje dle hodnoty, kterou napíšeme vlastnosti order. Tedy element, který má napsanou hodnotu 1 půjde před elementem, který má hodnotu 2 atd.

V kódu se to zapisuje takto:

order: 1;

Align-self

Pro úpravu zarovnání každého elementu individuálně použijeme vlasnost align-self. Této lze přiřadit takové hodnoty, jaké lze přiřadit align-items a jakékoli hodnoty přiřazené align-items přepíše.

Příklad:

.priklad {

display: flex;

height: 500px;

width: 500px;

background-color: grey;

}

 

.priklad-BOX1 {

background-color: blue;

width: 30%;

align-self: center;

}

 

.priklad-BOX2 {

background-color: red;

width: 30%;

align-self: flex-end;

}
